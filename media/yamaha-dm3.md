## General

- The "DM3" variant also contains a Dante card for extra routing, e.g. to stageboxes, while the "DM3S" (standard) does not.
- Firmware version 3 introduces an auto mixer, for podcasts, panel talks etc. It only works with the first 8 channels.

## Tasks

### Enter Maintenance Screen

1. Power off the unit.
1. Power on the unit while holding down the home button. Keep holding it until you arrive at the maintenance screen.

### Factory Reset

Do this to reset all data and configuration, other than the internal clock.

Enter the maintenance screen (se task) and select "Initialize All Memory". Alternatively select "Initialize Current Memory" to keep scene memory and libraries.

### Upload Custom Lock Screen Image

A custom image may be loaded into internal memory for use as the lockscreen.

Image requirements:

- Size: 800x480 pixels (aspect ratio 16:9.6)
- Format: PNG, JPG, BMP

Upload:

- Copy the file to the USB drive.
- Connect the USB drive.
- Go to the Save/Load settings.
- Select the image and press "load".
